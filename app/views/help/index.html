Create a project

"/>

To create a project, go to the Dashboard, or Your Projects page and click New Project. Enter a project name, description, due date, project visibility and include any collaborators you want to include in your project. You can always edit the details later.

Project Visibility:

Private:

If a project's visibility is set to Private, no external users may join the project or submit any feedback to the project and will not be shown on the Explore page or appear in search results unless you are member or owner of the project.

Unlisted:

If a project's visibility is set to Unlisted, external users may view project details and submit feedback to the project if they have the project link; however, the project will not be shown on the Explore page or appear in search results unless you are member or owner of the project.

Public:

If the project's visibilty is set to Public, external users can submit feedback to the project and the project will be listed on the Explore page and appear in search results for everyone.

Internal:

In the future, when GroupHub Professional is rolled out, marking a project as Internal will make your project visibility to only users of your organization (see Future Updates section below for more details). As of now, setting a project's visibility to Internal has the same effect as setting a project's visibility to Private.


Create a task

"/>

To create a task, go to the project's details, or the project's tasks page and click New Project. Enter a task name, a description, status, any tags, due date, the user the task should be assigned to and the milestone you want to include in your project. You can always edit the details later. You can also go to a milestone's detail page and create a task. Tasks created from the milestone's detail page will be automatically created under that milestone.

Search

"/>

In GroupHub, you can search by different criteria:


Search for a specific user by name:
If you are looking for a specific user, enter user: and the name of the user you are looking for.

Example: user:Jennifer Parker

Search for a specific project by name:
If you are looking for a specific project, enter project: and the name of the project you are looking for.

Example: project:Hill Valley Courthouse Square Refurbishment

Search for tasks that are assigned to a specific user:
If you are looking for tasks that are assigned to a specific user, enter assignedto: and the name of the user whose tasks you are looking for.

Example: assignedto: Marty McFly

Search for tasks by their status:
If you are looking for tasks with a specific status, enter status: and the status associated with tasks you are looking for.

Example: status:In progress

Search for tasks by their priority:
If you are looking for tasks with a specific priority, enter priority: and the priority associated with tasks you are looking for.

Example: priority:High

Search for tasks by their tags:
If you are looking for tasks with specific tags, enter tag: and the tag associated with tasks you are looking for.

Example: tag:Bug

Search for tasks by their due date:
If you are looking for tasks by their due date, enter duedate: and the due date for the tasks you are looking for.

Example: duedate:10/21/2015

Search for tasks by when the task was created:
If you are looking for tasks by the date it was created, enter datecreated: and the creation date.

Example: datecreated:11/13/1931

Search for tasks by when the task was last updated:
If you are looking for tasks by the date it was last updated, enter dateupdated: and the date last updated.

Example: dateupdated:11/5/1955

Search for tasks that belong to a specific milestone:
If you are looking for tasks that belong to a specific milestone, enter belongsto: and the name of the milestone the tasks belong to.

Example: belongsto:Clock Tower Restoration

Search for a specific milestone by name:
If you are looking for a specific milestone, enter milestone: and the name of the milestone you are looking for.

Example: milestone:Sisters of Mercy Soup Kitchen Renovation


Note: Only results you have the access to view will appaear in the search results.


Explore Projects

"/>

You can explore GroupHub projects by clicking on Explore in the navigation bar. Projects that are not the creator or collaborator on will have a Join option. You can click Join to request access to become a collaborator on the project. The creator of the project can decide whether to allow you to join the project. If the creator accepts your request, you will automatically be added to the list of collaborators on the project. If the creator rejects your request, you will not be added to the list of the collaborators at this time; however, you can request to join the project again.


Your Profile

"/>
"/>

Clicking on Your Profile or your name in the upper right corner of the navigation bar allows you to see the profile that other members see. This is the also the location where you can change your password. Click Edit on your profile page to change your name, email address, or password. Email address are shown by default; however, you can uncheck the "Show my email address" checkbox to hide your email address.

Future Updates


Public Feedback System

In a future update, end users will be able to submit feedback about your project including issues and/or requests. Projects that are set to private visibility will not have this feature enabled.


Additional Project Visibility Updates

In a future update, project creators will be able to make use of the internal visibility flag on their projects. Creators can set their projects to Internal. If a project is set to Internal, no external users outside their organization may join the project or submit any feedback to the project and will only be shown on the Explore page if the user is within the organization.


File Attachments

In a future update, users will be able to attach files to a specific task or comments that are associated with a specific task.
